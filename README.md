# Mowing Bidder Landing Page

Static, pre-launch landing page for Mowing Bidder built with Astro and the AstroWind template.

## Getting started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Things to customize

- `mailto:hello@example.com` in `src/components/FinalCTA.astro` and `src/components/Footer.astro`
- Pricing copy in `src/components/PricingTeaser.astro`
- Open Graph image at `src/assets/images/og.png`
