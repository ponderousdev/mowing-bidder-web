# Mowing Bidder Landing Page

Static pre-launch landing page for Mowing Bidder, built with Astro.

## Commands

```sh
npm install
npm run dev
npm run build
```

The dev server runs at `http://localhost:4321`.

## Project Structure

```text
public/
  og.png
src/
  components/
  layouts/
  pages/
```

## Things to customize

- Update the contact email in `src/components/FinalCTA.astro` and `src/components/Footer.astro`.
- Adjust pricing copy in `src/components/PricingTeaser.astro` and the demo formula in `src/components/DemoMock.astro`.
- Replace `public/og.png` with a branded Open Graph image.
